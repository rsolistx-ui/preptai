Infrastructure & Auth
Backend stability, OAuth fix, email system
Fix Security Improve
🔐
Google OAuth Reliability Fix
Replaced the single getSession() check on login with a dual-track system: onAuthStateChange listener fires the redirect the moment the OAuth token is processed, and getSession() handles returning users. Eliminated intermittent login failures where the session was missed due to timing.
✉️
Resend Email Integration
Wired RESEND_API_KEY for transactional email. Free-limit follow-up emails now send automatically when a free user hits their session limit. Contact form sends confirmation to user and notification to support@preptai.co.
⚡
JS Syntax Crash Fixed
A single extra closing brace in the showWelcomeMessage function was crashing the entire prept_v2.html script on load , industries never appeared, auth never ran. Fixed.
🏗
View Panel Layout Fix
Duo, Salary, Progress, and Video view panels were rendered outside the main layout container. All tabs now function correctly.
🛡
Security Headers
Added X-Content-Type-Options, X-Frame-Options, X-XSS-Protection, and Referrer-Policy headers via vercel.json across all routes.
📩
Contact Page with Jenn AI Assistant
Built full contact page at preptai.co/contact.html with form routing to support@preptai.co, country code dropdown for 33 countries, and Jenn , an AI support assistant powered by Claude , for instant 24/7 answers about the product.
